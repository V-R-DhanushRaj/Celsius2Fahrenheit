Celsius2Fahrenheit

A simple command-line program written in C that converts temperatures from Celsius to Fahrenheit.
This project was created as a foundational exercise to practice core C programming concepts such as input/output handling, arithmetic operations, and basic program structure.

IMPORTANT NOTICE:
No AI tools were used in the design or implementation of this project.
All logic and structure were written manually, with the primary learning resource being The C Programming Language book (Kernighan & Ritchie).

This project focuses on:
Understanding standard input/output in C
Basic mathematical transformations
